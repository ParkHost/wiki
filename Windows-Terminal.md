# Windows Terminal
## Installation:

- From (Windows) PowerShell (Core) as administrator:
```powershell
choco install microsoft-windows-terminal
```

## background-image:

![Alt Text](https://media.giphy.com/media/gjs3yK6xuIDIzVqryI/giphy.gif)
![sunrise.jpg](/sunrise.jpg =750x)

- Save your image/gif in the folder:
```cmd
'C:/Users/%username%/Documents/Windows Terminal/'
```
- The name of the file should be:
`universe.gif` 
***or***
`Sunrise.jpg`
or you have to edit it in the config file on line **19**
